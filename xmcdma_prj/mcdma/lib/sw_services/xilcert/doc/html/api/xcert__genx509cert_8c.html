Overview

This file contains the implementation of the interface functions for creating X.509 certificate for DevIK and DevAK public key.

This function creates the X.509 Certificate/Certificate Signing Request(CSR) 


Parameters

  

    
X509CertAddris the address of the X.509 Certificate buffer 

    
MaxCertSizeis the maximum size of the X.509 Certificate buffer 

    
X509CertSizeis the size of X.509 Certificate in bytes 

    
Cfgis structure which includes configuration for the X.509 Certificate.

  

  




Returns
    

  • XST_SUCCESS Successfully generated X.509 certificate/CSR
    • 

  • Error code In case of failure
    • 





Note
Certificate ::= SEQUENCE { tbsCertificate TBSCertificate, signatureAlgorithm AlgorithmIdentifier, signatureValue BIT STRING } 


Generate Sign Algorithm field


Calculate SHA2 Digest of the TBS certificate


Get the TBS certificate signature stored in Cert DB


If the Signature is available, compare the hash stored with the hash calculated to make sure nothing is changed. if hash matches, copy the stored signature else generate the signature again.


If the Signature is not available or Hash comparison from above fails, regenerate the signature.


Calculate signature of the TBS certificate using the private key


Copy the generated signature and hash into SignStore


Generate Signature field


Update the encoded length in the X.509 certificate SEQUENCE

This function stores the user provided value for the user configurable fields in the certificate as per the provided FieldType. 


Parameters

  

    
SubSystemIdis the id of subsystem for which field data is provided 

    
FieldTypeis to identify the field for which input is provided 

    
Valis the value of the field provided by the user 

    
Lenis the length of the value in bytes

  

  




Returns
    

  • XST_SUCCESS If whole operation is success
    • 

  • XST_FAILURE Upon any failure 
    • 





Look for the Subsystem Id. If it is there get the index and update the field of existing subsystem else increment index and add entry for new subsystem.
